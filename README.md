### ⚡ Engineering Focus

I lead a specialized team focused on high-performance digital ecosystems:

- **Bare-Metal AI Infrastructure:** Building and deploying LLMs locally on custom hardware (128GB RAM, advanced GPU arrays). No generic wrappers—just optimized, heavy-duty Python architectures running at the edge.
- **High-Frequency Mobile Apps:** Engineering complex marketplaces and ride-hailing platforms with React Native and TypeScript, handling real-time geolocation and heavy concurrent user loads. 
- **Kinetic UI Design:** Designing interfaces with flat geometric layering, subtle glassmorphism, and spring-driven kinetic physics. Zero latency, zero bloat.
- **Blockchain & Security:** Working natively in Linux/Ubuntu, building secure decentralized systems, handling applied cryptography, and algorithmic scanning of BIP39 mnemonic seeds.
