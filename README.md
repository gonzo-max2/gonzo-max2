### ⚡ What I Do

I lead a specialized team focused on high-performance digital ecosystems. Here is what I spend my time engineering:

- **Bare-Metal AI Infrastructure:** I build and deploy Large Language Models (LLMs) locally on custom hardware (128GB RAM, advanced GPU arrays). No generic API wrappers—just optimized, heavy-duty Python architectures that run at the edge.
- **High-Frequency Mobile Apps:** Using React, React Native, and TypeScript, I engineer complex marketplaces and ride-hailing platforms that handle real-time geolocation and heavy concurrent user loads. 
- **Kinetic UI Design:** I believe software should feel physical. I design interfaces with flat geometric layering, subtle glassmorphism, and spring-driven kinetic physics. Zero latency, zero bloat.
- **Blockchain & Security:** I work natively in Linux/Ubuntu, building secure decentralized systems, handling applied cryptography, and algorithmic scanning of BIP39 mnemonic seeds.
